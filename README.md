# 📚 Library Management System (LMS)

## 📌 Overview
This is a simple **Library Management System** built using **Python, Tkinter, and JSON**.
It provides a graphical user interface (GUI) to manage library books efficiently.

---

## 🚀 Features
- ➕ Add new books
- 📄 View all books
- ✏️ Update book details
- ❌ Delete books
- 🖱️ Click to auto-fill fields
- 💾 Data stored in JSON file

---

## 🛠️ Technologies Used
- Python 3
- Tkinter (GUI)
- JSON (Data storage)

---
